如何优雅的使用Windows之编辑器SublimeText

2015-02-27
技术

在茫茫人海中，为什么仅仅选择了Sublime作为主编辑器。

我的备选编辑器方案有四款。

  • vim 神之编辑器
  • emacs 编辑器之神
  • notepad++ 编辑器
  • 其他编辑器。

我的编辑器评判标准

语言方面要有HTML+CSS+JS+Python+autohotkey+markdown
兼顾可配置性和实用性。

其他编辑器有什么缺陷吗？
vim 符合终端时代的编辑模式，不符合GUI界面的编辑方法（可以键鼠操作），学习路径过于陡峭，插件配置过于恶心繁琐。
emacs 符合GUI界面的编辑方法，但是配置繁杂。
notepad++以前觉得非常方便，直到喜欢上了Sublime才知道，原来美观的东西也可以非常强大。
其他，使用的不多。

那么,这个Sublime究竟好在哪里？

  • 方便的Vim模式。
  • 各种语言特性支持。
  • 强大的社区支持。插件教程，balbalbala
  • 长的好看，而且能干。

如何优雅的使用Windows之普通人使用的软件

2015-02-25
技术

普通人使用的软件

我这篇是介绍一些软件，来方便平时使用电脑，减少在电脑上的时间。进而可以在省电时间陪陪女朋友。

首先，我要解释一下什么是普通人。抱歉，这里的普通人指的是使用过电脑，并且可能经常使用电脑但是并非电脑的狂热爱好者。

那么，按照这个标准，我就将这里的程序大致分为如下。

博客文章原创声明：
浏览器类

编辑器类

文字处理类

笔记软件类

多媒体类

通讯类

剪切板工具

浏览器类

浏览器分为四种，分别是chrome firefox opera ie
国产软件除了猎豹，其他不推荐。

我习惯使用Chrome，不仅仅是网页浏览速度会比其他浏览速度快上不少，而且还有更多的插件可以供你选择。

其他不介绍。

对Chrome比较感兴趣的可以查看我的另一篇关于Google出品必属精品的文章。

编辑器类

这里是编辑器，为什么要使用编辑器？我们普通人并不是码农啊？

很抱歉，你真的需要使用编辑器。因为不管你怎么使用Word或者其他需要大量输入文字的时候，在Word等等软件里面进行批量修改的。

请注意，Word是文字处理软件，但是其他文本编辑器是是文字编辑软件。编辑好，然后专注于排版。这样才是最好。

文字处理类

Office系列，理由：微软出品绝对是流弊。

笔记软件类

为知笔记。将平时的文本放到云端，这样就可以避免一大片的文档。而且，Wiz笔记可以搜索文件内内容，可以在文档类进行各种强力的编辑，更加恐怖的是其中的文档制作工具和各种扩展。

多媒体类

QQ影音，我一般情况下都是下载下来以后播放。对在线的东西一般情况下不感兴趣

聊天

QQ international 弹窗少。

everything

我之前推荐过，你可以在上一篇文章中看到。使用得当，可以省下你很多的时间。

如何优雅的使用Windows之文件以及文件夹规范

2015-02-24
技术

　系列之一：官方的基本快捷操作与你的文件分布应该是什么样子的。

首先，请反省一下自己。是不是桌面很乱，文件经常找不到？

其次，如果你使用Windows7以上的系统，反省一下，是不是经常每个盘根目录下面分类超过六个？

好，想到这里，我们就提出解决方法来。

我们把问题仔细分解一个一个来处理。

第一个，桌面乱，程序图标多

  1. 程序除非是绿色软件，或者开发环境，否则全部装在C盘的program fies里面。并且为软件创建快捷方式并命名为简写，比如xmind 简写为xm 为知笔记简写为wz ，新建一个文件夹，把快捷方式放到这里来。并且为这个文件夹配置好环境变量。我是将D:\Sysconfig存放在环境变量之中，这样就可以直接运行win+R 输入xm 回车，便可以打开xmind程序了。**这是为了把桌面上的无用的图标放在这里
  2. 这样的话，你就只需要管理那个文件夹就可以管理程序的运行了。

是不是很方便？

第二个,桌面乱，文件与文件夹多

1.这个好解决，推荐使用一款工具叫做Everything，这款软件只需要你好好的命名文件与文件夹，便可以秒搜。什么意思？比如我有一个文件 名字叫做 20150215如何优雅的使用windows之文件与文件夹规范，但是时间长了我忘了放在哪里，而且我只记得 2015年写的，关于文件夹的，因为写的是Blog，所以用的肯定是markdown文件，以md结尾,这个时候 我打开Everything，输入md 2015 文件，瞬间搜索到这个文件。Everything被称为文件搜索类程序中的小李飞刀。天下武功，无坚不摧，唯快不破。就是这个道理。你不需要知道文件在哪，你只要记得文件怎么命名的就好。只是这个软件对使用者有一定的命名意识。一定要有一定的命名规范。

2.好好分类，不要随手建立一个新建文件夹或者随手建立一个新建文件然后连名字都不写。在几个小时后你就没有办法找到你要的文件了。这个时候你去一个一个打开看看？

第三个,文档怎么分类命名？

1.我认为，对于文档，文件名中应该包含 时间+文件标题或者几个字概括内容+是否完成。比如 20150215记录文件夹灵感Undo.md 比如 20150215常州恐龙园门前合照.jpg 或者没必要命名时间的一些资源文件：AndroidButtonIOS风格.psd
2.对于文件夹，我个人倾向于使用英语。比如File Document Workspace ;我感觉我的文件可能不算非常的多，所以文件夹的命名并没有什么特点。比如我的正在编写的Android代码就在D:\Workspace\Android 里面。而绝大部分对排版要求不是非常高的我都是存放在为知笔记里面。

下面是我的关于分区的看法，我们将硬盘分为四个区.

  • C盘安装一些必须运行的环境,比如JDK 微软的运行库 补丁还有一些其他的非绿色的软件.我单独给了一个根文件夹用于存放安装的环境。叫做DevEnv
  • D盘安放一些IDE程序,编写的Office文档以及暂时的电子书文件。
  • E盘安放一些用来休闲娱乐和已经保存下来的资料,游戏，以及下载区和云同步的文件。
  • F盘安放影视文件，以及私人的图片文件。

这样分区是为了更好的区分自己的状态,不要在学习的时候去其他的盘符里面.

Chorthotkey、Candy、VimDesktop 三剑客

2015-02-22
技术

概要介绍

经过长时间的电脑的使用，我渐渐的喜欢上了autohotkey这个神器，喜欢Python是因为Python非常的强大，喜欢AHK则是因为这个可以给我的平常生活带来太多的方便。

长时间的使用，毕竟也积攒了一些脚本，加上对很多脚本的收集以及修改，勉强算得上是一个入门的AHK菜鸟了。

AHK的优点可以对快捷键进行编程，对一个经常使用电脑自动化处理一些事情的人说，非常之方便，但缺点，太多,效率低，语法乱，数据结构乱。让我写一段文字处理简直就让我非常难受。

但为了取长补短，我仅仅使用其中的对快捷键编程的功能，加上收集来的别人的一些函数。修改集成，然后做一个顺手的AHK脚本，名字就叫做Chorthotkey。开源，协议选择Mit协议。简而言之也就是你改了名字版权就是你的了。

我在编写这篇文章的时候，也在重构并进行V0.4版本的完善。

但是先讲讲什么使用AHK编写的脚本比较好。

作为一个经常使用电脑的人，什么时候我们应该使用AHK？

我觉得是，当这个功能不值得用另一个大的软件代替的时候，或者快捷键编程的时候。
先说第一点：当我想使用一个功能的时候，比如快速打开一个网页，或者进行文字上的简单处理，这个时候就没有必要单独下载一个软件进行处理，这个在AHK中就是简单的几行代码就可以搞定。
比如打开网站直接 Run www.baidu.com AHK就会直接调用相关浏览器打开baidu.这么简单的东西没有必要单独下载一个软件进行管理。
再说第二点：当我想着对快捷键进行编程调用进行自动化处理东西的时候，或者编写一套自己的热键用来打游戏或者其他的一些编辑用途等等，都可以。理论上来说，AHK几乎是只有想不到没有做不到的。

那么，AHK不能替代什么东西？
在文件管理方面，尽管你可以自己编写很多的快捷键，但你没有办法超越TotalCMD，没有办法超越Listary或者Everything，所以，你编写的东西只需要辅助你使用这些东西就好了。
在文本编辑方面，你始终无法超越sublime text 3，无法超越vi，无法超越Emacs，你只需要辅助你使用这些工具即可。
同理。

既然说了AHK那么多的不足和缺憾，但是AHK在我的脚本语言中依然傲立第二名，就是因为它擅长的快捷键编程。

好了，废话这么多，总该说点干货了。下面是三个脚本

  • Candy 一个非常强大的第三方弹出菜单amii编写。
  • ChortHotKey 一个我编写的工具
  • vimd 无所不在的vim党编写的软件。

新的一年的计划

2015-02-20
计划


老实来说，我并不喜欢在新的一年将自己的计划发表在社交网络上面，因为这样仿佛会在发表计划的那一瞬间得到非常大的满足感，这样并不利于计划的实现。

但是，在这里发表并不能算是在社交网络发表，认认真真的写下来，只是希望我可以在今后的时间里好好的专注于这些目标，而不是乱七八糟的学习。

1.每天锻炼45分钟。
2.坚持平均学英语的时间达到30min。
3.坚持每天Coding和坚持记笔记整理成Blog
4.把我的Srapy+Django+Android融会贯通，写一个大的项目发表到Github上面，并维护半年
5.今年结束之前争取可以拿下两个大奖和并且到一个公司实习。
